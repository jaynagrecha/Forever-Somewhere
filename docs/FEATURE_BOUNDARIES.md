# Feature boundaries — Forever Somewhere

| Section | Role | Unique tools |
|---------|------|--------------|
| **Moments** | Past — what happened | Tags, photos, playlists, milestones, timeline |
| **Somewhere** | Map — where it happened | Auto from memories + trip pins |
| **Someday** | Future — what we want | Budget, checklist, promote → map, log → memory |
| **Forever** | Heart — words | Time capsules (locked) + love notes (instant) |
| **Settings** | Practical | Export, custom anniversaries, PWA install |

Bridges: Someday → Map (promote) · Someday → Moments (log as memory) · Moments → Map (auto pins)
